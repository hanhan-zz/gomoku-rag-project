# Gomoku Rules 五子棋规则

## 1. Basic Rules 基本规则

### Definition 定义
五子棋是在 15×15 棋盘上进行的两人对弈游戏，双方轮流在空白交叉点落子。通常黑棋先手，白棋后手。  
在本项目中，默认：
- 黑棋（BLACK）为人类玩家
- 白棋（WHITE）为 AI 玩家

### Core Rule 核心规则
每一步只能下在一个**空白位置**。  
如果某个位置已经有棋子，则该步为非法落子。  
游戏在以下情况之一结束：
1. 某一方形成连续五子
2. 棋盘已满且无人获胜

### Why It Matters 为什么重要
这是系统进行合法性判断、前端交互验证和后端胜负判定的基础。  
所有后续的问答、复盘和策略分析都建立在这些规则之上。

### Example Explanation Sentence 可用于问答/复盘的解释句
- 这一步首先必须满足五子棋的基本合法性要求，即只能落在空白位置上。  
- 当前系统判断该位置可落子，是因为该点未被占用且游戏尚未结束。

## 2. Win Conditions 胜负条件

### Definition 定义
当某一方在横向、纵向或任意一个对角线方向形成**连续五个同色棋子**时，该方立即获胜。

### Winning Directions 连珠方向
获胜可能来自四类方向：
- 水平
- 垂直
- 主对角线
- 副对角线

### Draw Condition 平局条件
如果棋盘已满，且双方都没有形成连续五子，则判定为平局。

### Why It Matters 为什么重要
系统中的胜负判断直接决定：
- 对局是否结束
- 最终赢家是谁
- 是否需要触发赛后复盘
- 是否需要高亮 winning line

### Example Explanation Sentence 可用于问答/复盘的解释句
- 这一步之所以关键，是因为它在对角线方向上完成了五连，从而直接结束比赛。  
- 这一步没有立即获胜，但它明显接近形成连续五子，因此具有较高战略价值。

## 3. Common Terms 常见术语

## 3.1 Empty Point 空点
棋盘上尚未被落子的交叉点。

## 3.2 Line 线
若干连续位置构成的方向线，可为水平、垂直或对角线。

## 3.3 Threat 威胁
某一步虽然不立即获胜，但为后续形成更强攻势创造条件，这种局面可称为威胁。

## 3.4 Block 封堵
通过落子阻止对手继续延伸连珠或形成更强棋形。

## 3.5 Offensive Move 进攻性落子
以扩展己方棋形、制造威胁、争取主动为主要目的的落子。

## 3.6 Defensive Move 防守性落子
以阻止对手形成威胁或立即获胜为主要目的的落子。

### Why It Matters 为什么重要
这些术语会在：
- Move QA
- Review
- RAG retrieval
中频繁出现，因此必须定义清楚并保持表述一致。

### Example Explanation Sentence 可用于问答/复盘的解释句
- 这一步更偏向防守性落子，因为它主要作用是封堵对手的延伸方向。  
- 当前局面的关键不在于立即进攻，而在于先处理对手已经形成的威胁。

## 4. Notes for AI Explanation 面向 AI 解释的说明

为了让大模型在问答和复盘中给出更稳定的回答，可以优先使用以下解释框架：

1. 先说明这一步是否合法  
2. 再说明这一步的主要意图：进攻 / 防守 / 平衡  
3. 再说明它影响了哪条线或哪类威胁  
4. 最后给出局面上的战略意义

### Recommended Explanation Pattern 推荐解释模板
- 这一步是合法落子，并且其主要意图是____。  
- 它影响的关键方向是____。  
- 从局面上看，这一步的意义在于____。
